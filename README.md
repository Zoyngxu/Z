
📚 刷题宝（ShuaTiBao）

打工人的记忆外挂，专治各种考试焦虑。

这是什么？

刷题宝是一款轻量级、本地运行、开箱即用的刷题工具。

它诞生于一个非常朴素的想法：

为什么上班已经够累了，下班还要背题？

从安全考试到岗位认证，从公司题库到培训测试，从月考到年考，从应知应会到领导突然抽查——总有做不完的题，背不完的知识点。

于是，刷题宝出现了。

⸻

🚀 为什么开发它？

因为很多企业内部题库：

* 没有刷题系统
* 只有 Excel
* 只有 Word
* 或者压根是一堆截图

员工只能：

打印
↓
背
↓
忘
↓
再背
↓
再忘

形成完美闭环。

刷题宝的目标只有一个：

让背题这件事，稍微没那么痛苦。
